## Visión general creada por IA

In JavaScript, `?`and `??`represent distinct operators:

1\. `?`(Conditional Ternary Operator)

The `?`symbol is part of the conditional (ternary) operator, which provides a concise way to write an `if...else` statement in a single line. It takes three operands: a condition, an expression to execute if the condition is true, and an expression to execute if the condition is false.

Syntax:

JavaScript

```
condition ? expressionIfTrue : expressionIfFalse;
```

Example:

JavaScript

```
let age = 20;let status = (age >= 18) ? "Adult" : "Minor"; // status will be "Adult"
```

2\. `??`(Nullish Coalescing Operator)

The `??`symbol represents the nullish coalescing operator, introduced in ECMAScript 2020. This operator provides a way to assign a default value to a variable if the variable's value is `null` or `undefined`. It differs from the logical OR operator (`||`) by only considering `null` and `undefined` as "falsy" values, whereas `||` considers other falsy values like `0`, `''` (empty string), and `false` as well.

Syntax:

JavaScript

```
leftHandSide ?? rightHandSide;
```

Example:

JavaScript

```
let username = null;let displayUsername = username ?? "Guest"; // displayUsername will be "Guest"let count = 0;let defaultCount = count ?? 10; // defaultCount will be 0 (because 0 is not null or undefined)
```
