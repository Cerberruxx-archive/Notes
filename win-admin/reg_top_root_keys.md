| Key	| For	| Use Case|
|-------|-------|---------|
|HKEY_CLASSES_ROOT (HKCR)| File associations |Tells Windows which app opens a .pdf or .txt file |
|HKEY_CURRENT_USER(HKCU) |Your settings |Changing wallpaper, taskbar, app themes |
|HKEY_USERS (HKU) |All user profiles |Mostly used by admin to manage multiple accounts |
|HKEY_CURRENT_CONFIG (HKCC) |Hardware profile |Info about current display/printers |
|HKEY_LOCAL_MACHINE (HLM) |System settings |Hardware drivers, boot settings and security
